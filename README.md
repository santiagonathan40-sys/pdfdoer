# PDFDoer Frontend

A modern PDF tools frontend built with React, Vite, TypeScript, and Tailwind CSS.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Environment

Copy `.env.example` to `.env.local` and set the backend URL when the API is ready.

```env
VITE_API_BASE_URL=http://localhost:8000
```
